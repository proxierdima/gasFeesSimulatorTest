БЫСТРЫЙ СТАРТ:

1. Установить зависимости:
   pip install -r requirements.txt
   npm install

2. Настроить .env файл:
   HARNESS_PRIVATE_KEY=0xВАШ_КЛЮЧ

3. Запустить:
   python main.py --project-root "G:\glh_v4" --password 12345678

ОЖИДАЕМОЕ ВРЕМЯ:
   - Развертывание контракта: ~10 секунд
   - Подтверждение транзакции: ~5-10 минут
   - Полный тест: ~10-15 минут
